# Testnet Assets

## Leverage Testnet Assets

Blockchain development requires extensive development and testing before deploying applications to the mainnet. To facilitate this process, Venly provides testnet assets to new users who sign up on the [Portal](https://portal.venly.io/?utm_source=website&utm_medium=docs&utm_campaign=General+-+Venly+Internal). These assets allow developers and non-developers to explore blockchain functionalities via APIs or a user-friendly interface.

---

## Start Testing Right Away!

When you sign up with Venly, you gain instant access to a suite of testnet assets designed to help you explore and integrate blockchain functionalities effortlessly. These assets are automatically created and deposited into your Venly wallet, allowing you to test our API endpoints immediately and experience the full potential of Venly's platform.

## What You Receive?

When you sign up on the [Portal](https://portal.venly.io/?utm_source=website&utm_medium=docs&utm_campaign=General+-+Venly+Internal) or create a new company, your account is initialized with the following testnet assets:

### 1. Testnet User & Wallet

<Steps>
  <Step title="Testnet User Created">
    A **testnet user** is created for you automatically
  </Step>
  
  <Step title="Polygon Wallet Linked">
    A **Polygon testnet wallet** is created and linked to your testnet user
  </Step>
  
  <Step title="Primary Address">
    This wallet is your primary address for accessing the preloaded testnet tokens
  </Step>
</Steps>

## Frequently Asked Questions

<AccordionGroup>
  <Accordion title="Are testnet assets available for all new users?" icon="circle-question">
    Yes, all new users who sign up on the Venly Portal automatically receive testnet assets to start testing immediately.
  </Accordion>
  
  <Accordion title="Can I request more testnet tokens?" icon="plus-circle">
    Yes, you can request additional POL tokens from the [Venly Faucet](/getting-started/faucets) once every 24 hours.
  </Accordion>
  
  <Accordion title="Can I create additional NFT contracts?" icon="file-contract">
    Absolutely! You can create as many NFT contracts as you need using the NFT API.
  </Accordion>
  
  <Accordion title="Do testnet assets have real value?" icon="circle-dollar">
    No, testnet assets have no real-world monetary value. They are solely for testing purposes.
  </Accordion>
  
  <Accordion title="Can I use testnet assets on mainnet?" icon="network-wired">
    No, testnet assets only work on testnets. You'll need real tokens for mainnet deployments.
  </Accordion>
  
  <Accordion title="What happens if I run out of testnet tokens?" icon="battery-empty">
    You can request more POL tokens from the Faucet or contact support if you need additional testing resources.
  </Accordion>
</AccordionGroup>
