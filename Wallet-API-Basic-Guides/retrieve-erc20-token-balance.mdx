ERC-20 token balances include all smart contract tokens like USDC, USDT, DAI, and thousands of other tokens across supported blockchains.
</Info>

## Overview

Checking ERC-20 token balances is essential for:
- **Portfolio management**: Tracking all token holdings across wallets
- **DeFi operations**: Verifying token balances before swaps or lending
- **Payment validation**: Confirming token receipts and balances
- **Application logic**: Making decisions based on specific token holdings

<Tip>
Use the `includePossibleSpam` parameter to filter out potential spam tokens and focus on legitimate holdings.
</Tip>

## Prerequisites

To retrieve ERC-20 token balances, you need either:
- A valid wallet ID from your Venly-managed wallets
- OR a blockchain address and the corresponding network type
- Optionally, a specific token contract address for targeted queries

## Method 1: Get All Tokens by Wallet ID

Perfect for getting a complete overview of all ERC-20 tokens in a Venly-managed wallet.

### API Reference

#### Endpoint
```http
GET /api/wallets/{walletId}/balance/tokens
```

#### Parameters

| Parameter | Type | Description | Required | Example |
|-----------|------|-------------|----------|---------|
| `walletId` | Path | The unique ID of the wallet | ✅ | `"f21dc1e5-4253-487e-8927-3b5247437f99"` |
| `includePossibleSpam` | Query | Include potential spam tokens | ❌ | `false` |

### Code Examples

<CodeGroup>

```http Get All Tokens (Exclude Spam)
GET /api/wallets/f21dc1e5-4253-487e-8927-3b5247437f99/balance/tokens?includePossibleSpam=false
```

```http Get All Tokens (Include All)
GET /api/wallets/f21dc1e5-4253-487e-8927-3b5247437f99/balance/tokens
```

</CodeGroup>

## Method 2: Get All Tokens by Address

Use this method to check any wallet address, including external wallets not managed by Venly.

### API Reference

#### Endpoint
```http
GET /api/wallets/{secretType}/{walletAddress}/balance/tokens
```

#### Parameters

| Parameter | Type | Description | Required | Example |
|-----------|------|-------------|----------|---------|
| `secretType` | Path | The blockchain network | ✅ | `"ETHEREUM"` |
| `walletAddress` | Path | The wallet's blockchain address | ✅ | `"0x9f0847430a41ff52573857b49A87076e1748ed6A"` |
| `includePossibleSpam` | Query | Include potential spam tokens | ❌ | `false` |

### Code Examples

<CodeGroup>

```http Get Ethereum Tokens
GET /api/wallets/ETHEREUM/0x9f0847430a41ff52573857b49A87076e1748ed6A/balance/tokens
```

```http Get Polygon Tokens
GET /api/wallets/MATIC/0x742d35Cc6634C0532925a3b8D4C9db96C4b4d8b9/balance/tokens
```

```http Get Arbitrum Tokens (No Spam)
GET /api/wallets/ARBITRUM/0x9f0847430a41ff52573857b49A87076e1748ed6A/balance/tokens?includePossibleSpam=false
```

</CodeGroup>

## Method 3: Get Specific Token by Wallet ID

Check the balance of a particular token in a Venly-managed wallet.

### API Reference

#### Endpoint
```http
GET /api/wallets/{walletId}/balance/tokens/{tokenAddress}
```

#### Parameters

| Parameter | Type | Description | Required | Example |
|-----------|------|-------------|----------|---------|
| `walletId` | Path | The unique ID of the wallet | ✅ | `"f21dc1e5-4253-487e-8927-3b5247437f99"` |
| `tokenAddress` | Path | Contract address of the token | ✅ | `"0xA0b86a33E6441E8C8C7014A37712A4F2BD4C6a9"` |

### Code Examples

<CodeGroup>

```http Check USDC Balance
GET /api/wallets/f21dc1e5-4253-487e-8927-3b5247437f99/balance/tokens/0xA0b86a33E6441E8C8C7014A37712A4F2BD4C6a9
```

```http Check UNI Balance
GET /api/wallets/f21dc1e5-4253-487e-8927-3b5247437f99/balance/tokens/0xfa7f8980b0f1e64a2062791cc3b0871572f1f7f0
```

</CodeGroup>

### Response Fields

| Field | Description |
|-------|-------------|
| `tokenAddress` | Contract address of the token |
| `rawBalance` | Balance in smallest token unit |
| `balance` | Human-readable balance amount |
| `decimals` | Number of decimal places |
| `symbol` | Token symbol (UNI, USDC, etc.) |
| `name` | Full token name |
| `logo` | Token logo URL |
| `type` | Token standard (ERC20, ERC721, etc.) |
| `transferable` | Whether the token can be transferred |
| `exchange.usdPrice` | Current USD price per token |
| `exchange.usdBalanceValue` | Total USD value of holdings |
| `categories` | Token categories/tags |
| `links` | Official project links |
| `portfolioPercentage` | Percentage of total portfolio value |

<Warning>
Use `rawBalance` for precise calculations to avoid floating-point precision issues. The `balance` field is for display purposes only.
</Warning>

## Spam Token Filtering

<Accordion title="Understanding Spam Tokens">

**What are Spam Tokens?**
- Tokens sent to wallets without permission
- Often used for marketing or malicious purposes
- Can clutter wallet interfaces and confuse users

**Filtering Options:**
- `includePossibleSpam=false` - Exclude suspected spam tokens
- `includePossibleSpam=true` - Include all tokens (default)
- Omit parameter - Include all tokens

**Best Practice:**
Use `includePossibleSpam=false` for user-facing applications to provide a cleaner experience.

</Accordion>

```json

  "success": false,
```

**Network Unavailable**
```json
{

  "message": "Unable to fetch token data from the blockchain network"
}
```
