## Step 1: Upload Your Business Icon

### API Endpoint

<CodeGroup>
```javascript JavaScript
const formData = new FormData();
formData.append('icon', iconFile); // iconFile is a File object

const response = await fetch('/account/settings/branding/icon', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: formData
});

const data = await response.json();
console.log('Icon uploaded:', data.id);
```

```python Python
import requests

files = {'icon': open('your-logo.jpg', 'rb')}

response = requests.post(
    '/account/settings/branding/icon',
    headers={'Authorization': 'Bearer YOUR_API_KEY'},
    files=files
)

data = response.json()
print('Icon uploaded:', data['id'])
```

```bash cURL
curl -X POST '/account/settings/branding/icon' \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -F 'icon=@your-logo.jpg'
```
</CodeGroup>

### Request Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `icon` | File | ✅ | Square image file (minimum 128px × 128px) |

<Tip>
**Image Guidelines:**
- Use square dimensions (1:1 aspect ratio)
- Minimum size: 128px × 128px
- Recommended size: 512px × 512px for best quality
- Supported formats: JPG, PNG, GIF, SVG
- Keep file size under 5MB for optimal performance
</Tip>

## Step 2: Configure Checkout Branding

### API Endpoint

<CodeGroup>
```javascript JavaScript
const iconId = "file_1Nk6WcFdbOilQPiZLgnem6MT"; // From previous step

const response = await fetch('/account/settings/branding', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    settings: {
      branding: {
        primary_color: "#BF40BF",
        secondary_color: "#000000",
        icon: iconId
      }
    }
  })
});

const data = await response.json();
console.log('Branding configured:', data);
```

```python Python
import requests

icon_id = "file_1Nk6WcFdbOilQPiZLgnem6MT"  # From previous step

response = requests.post(
    '/account/settings/branding',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'settings': {
            'branding': {
                'primary_color': '#BF40BF',
                'secondary_color': '#000000',
                'icon': icon_id
            }
        }
    }
)

data = response.json()
print('Branding configured:', data)
```

```bash cURL
curl -X POST '/account/settings/branding' \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
    "settings": {
      "branding": {
        "primary_color": "#BF40BF",
        "secondary_color": "#000000",
        "icon": "file_1Nk6WcFdbOilQPiZLgnem6MT"
      }
    }
  }'
```
</CodeGroup>

### Request Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `settings.branding.primary_color` | string | ✅ | Hex color code for the left side background |
| `settings.branding.secondary_color` | string | ✅ | Hex color code for the PAY button |
| `settings.branding.icon` | string | ✅ | Icon ID from the upload response |

<Tip>
**Color Guidelines:**
- Use hex color codes (e.g., #BF40BF, #000000)
- Primary color appears on the left side of checkout
- Secondary color is used for the PAY button
- Ensure good contrast for accessibility
- Test colors on different devices and screens
</Tip>

## Integration Examples

<CodeGroup>
```javascript JavaScript
// Complete branding setup workflow
async function setupCheckoutBranding(logoFile, primaryColor, secondaryColor) {
  try {
    // Step 1: Upload icon
    const formData = new FormData();
    formData.append('icon', logoFile);
    
    const uploadResponse = await fetch('/account/settings/branding/icon', {
      method: 'POST',
      headers: { 'Authorization': 'Bearer YOUR_API_KEY' },
      body: formData
    });
    
    const uploadData = await uploadResponse.json();
    const iconId = uploadData.id;
    
    console.log('✅ Icon uploaded:', iconId);
    
    // Step 2: Configure branding
    const brandingResponse = await fetch('/account/settings/branding', {
      method: 'POST',
      headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        settings: {
          branding: {
            primary_color: primaryColor,
            secondary_color: secondaryColor,
            icon: iconId
          }
        }
      })
    });
    
    const brandingData = await brandingResponse.json();
    console.log('✅ Branding configured successfully');
    
    return { iconId, branding: brandingData };
    
  } catch (error) {
    console.error('❌ Branding setup failed:', error);
    throw error;
  }
}

// Usage
const logoFile = document.getElementById('logo-input').files[0];
const result = await setupCheckoutBranding(logoFile, '#1E40AF', '#3B82F6');
```

```python Python
import requests

def setup_checkout_branding(logo_path, primary_color, secondary_color):
    """Complete branding setup workflow"""
    try:
        # Step 1: Upload icon
        with open(logo_path, 'rb') as logo_file:
            files = {'icon': logo_file}
            upload_response = requests.post(
                '/account/settings/branding/icon',
                headers={'Authorization': 'Bearer YOUR_API_KEY'},
                files=files
            )
        
        upload_data = upload_response.json()
        icon_id = upload_data['id']
        
        print(f'✅ Icon uploaded: {icon_id}')
        
        # Step 2: Configure branding
        branding_response = requests.post(
            '/account/settings/branding',
            headers={
                'Authorization': 'Bearer YOUR_API_KEY',
                'Content-Type': 'application/json'
            },
            json={
                'settings': {
                    'branding': {
                        'primary_color': primary_color,
                        'secondary_color': secondary_color,
                        'icon': icon_id
                    }
                }
            }
        )
        
        branding_data = branding_response.json()
        print('✅ Branding configured successfully')
        
        return {'icon_id': icon_id, 'branding': branding_data}
        
    except Exception as error:
        print(f'❌ Branding setup failed: {error}')
        raise error

# Usage
result = setup_checkout_branding('logo.png', '#1E40AF', '#3B82F6')
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Icon upload failed">
    **Common causes:**
    - Image not square (must be 1:1 aspect ratio)
    - Image too small (minimum 128px × 128px)
    - Unsupported file format
    - File size too large
    
    **Solution:** Ensure your image meets all requirements: square dimensions, minimum size, supported format (JPG, PNG, GIF, SVG).
  </Accordion>
  
  <Accordion title="Invalid color format">
    **Error:** Color code not recognized
    
    **Causes:**
    - Missing # symbol
    - Invalid hex characters
    - Wrong color format
    
    **Solution:** Use proper hex format: #RRGGBB (e.g., #1E40AF, #000000).
  </Accordion>
  
  <Accordion title="Branding not appearing">
    **Possible issues:**
    - Incorrect icon ID
    - Branding not applied to checkout session
    - Cache issues
    
    **Solution:** Verify icon ID, clear browser cache, and ensure branding is properly configured.
  </Accordion>
  
  <Accordion title="Colors look different">
    **Common causes:**
    - Monitor color calibration
    - Different color profiles
    - Browser rendering differences
    
    **Solution:** Test on multiple devices and browsers, use standard hex colors, consider accessibility guidelines.
  </Accordion>
</AccordionGroup>
