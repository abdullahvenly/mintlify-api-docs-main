## Two-Factor Authentication (2FA)

Protects your account even if someone obtains your password.

**Setup:**
1. Download Google Authenticator ([iOS](https://apps.apple.com/app/google-authenticator/id388497605) | [Android](https://play.google.com/store/apps/details?id=com.google.android.apps.authenticator2))
2. Scan QR code during registration
3. Save backup code securely
4. Enter 6-digit code to verify

<Warning>
  Save your 2FA secret key. Without it, you'll lose account access if device is lost.
</Warning>

## Account Verification

**Bank Accounts:** Email verification (click link within 24 hours)

**Crypto Wallets:**
- Managed: Instant, Venly-secured
- External: Send 1 USDC from registered wallet

## Transaction Security

**On-Ramp:** Include reference code in bank transfer

**Off-Ramp:** Wallet debited immediately (cannot cancel)

**Both:** All transactions compliance-monitored

## If Account Compromised

  <Step title="Immediate Action">

  <Step title="Document">
