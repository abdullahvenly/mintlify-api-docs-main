## Dashboard Layout

Your dashboard provides an overview of your account status and quick access to key features.

## Account Status

Your account status is displayed at the top:

- **Processing**: Under verification - can whitelist accounts, cannot create transactions
- **Verified**: Full access - can create all transactions

## Main Navigation

| Menu Item | Purpose |
|-----------|---------|
| **Dashboard** | Overview and quick actions |
| **Accounts** | Manage bank accounts and wallets |
| **Transactions** | View and create transactions |
| **Fees** | View fee structure |

## Quick Actions

Once verified, use action buttons to:
- Create on-ramp transactions
- Create off-ramp transactions

<Note>
  Quick actions are only available after account verification
</Note>

## Overview Sections

### Accounts
View your whitelisted:
- Bank accounts with status
- Crypto wallets with status

### Recent Transactions
See recent transaction activity with:
- Transaction type
- Status
- Amounts

### Fee Information
Quick view of current fee structure
