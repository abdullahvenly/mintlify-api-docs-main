## Dashboard Layout

Your dashboard provides an overview of your account status, metrics, and quick actions.

## Key Metrics

<CardGroup cols={2}>
  <Card title="Total Volume" icon="trending-up">
    Your 30-day transaction volume (determines fee tier)
  </Card>
  <Card title="Active Accounts" icon="wallet">
    Number of verified bank accounts and wallets
  </Card>
  <Card title="Pending Requests" icon="clock">
    Transactions awaiting processing
  </Card>
  <Card title="Completed" icon="check-circle">
    Successfully processed transactions
  </Card>
</CardGroup>

## Business Status Badge

Located at the top right:

- **Yellow Badge** (Application in Process): Verification pending
- **Green Badge** (Approved Business): Full access granted

## Navigation Menu

Access features from the sidebar:

| Menu Item | Purpose |
|-----------|---------|
| **Dashboard** | Main overview |
| **Accounts** | Manage bank accounts and wallets |
| **Transactions** | Create and monitor transactions |
| **Yield** | Earn returns on stablecoins |
| **Fees** | View pricing structure |
| **Settings** | Account configuration |

## Transaction Statuses

| Status | Color | Meaning |
|--------|-------|---------|
| Transaction Created | Blue | Just initiated |
| Waiting for Funds | Amber | Awaiting bank transfer |
| Processing | Purple | Being processed |
| Completed | Green | Successfully finished |
| Failed | Red | Error occurred |
