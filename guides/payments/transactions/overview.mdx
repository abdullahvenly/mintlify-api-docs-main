## Transaction Flow

**On-Ramp:** Create transaction → Transfer from bank → Crypto arrives in wallet

**Off-Ramp:** Create transaction → Wallet debited instantly → Fiat arrives in bank

## Transaction Statuses

| Status | Meaning | Your Action |
|--------|---------|-------------|
| **Transaction Created** | Just initiated | Complete bank transfer (on-ramp) |
| **Waiting for Funds** | Awaiting payment | None - system monitoring |
| **Processing** | Being processed | None - wait for completion |
| **Completed** | Successfully finished | Funds available |
| **Failed** | Error occurred | Contact support |

## Prerequisites

Before creating transactions:

✅ Business approved  
✅ Bank account verified  
✅ Crypto wallet active  
✅ Sufficient funds

## Fees

**On-Ramp:** 0.5% processing + 0.30-0.70% currency tier  
**Off-Ramp:** 1.0% processing + 0.30-0.70% currency tier

[View detailed fees →](/guides/payments/fees)

## Reference Codes

**On-Ramp:** You receive a reference code (e.g., `ON-ABC12345`) - **must include in bank transfer**

**Off-Ramp:** You receive a reference code (e.g., `OFF-XYZ67890`) - for tracking only

<Warning>
  For on-ramps, forgetting the reference code will delay processing by 1-2 days
</Warning>

## Processing Times

**On-Ramp:**
- 0-60 min: Complete bank transfer
- 60-120 min: Crypto sent to wallet

**Off-Ramp:**
- 0 min: Wallet debited
- 24-48 hrs: Fiat in bank
