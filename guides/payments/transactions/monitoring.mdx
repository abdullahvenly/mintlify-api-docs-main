## Transaction Dashboard

View and monitor all your transactions from the Transactions page.

<Info>
  Access: Click **Transactions** in the main navigation menu
</Info>

## Transaction Views

### Pending Transactions

Shows active transactions requiring action or in progress:

- **Transaction Created**: Awaiting your bank transfer
- **Waiting for Funds**: Bank transfer pending confirmation
- **Processing**: Being processed by the platform

### Processed Transactions

Shows completed transaction history:

- **Completed**: Successfully finished
- **Failed**: Encountered an error

## Transaction Card Information

Each transaction displays:

| Field | Description |
|-------|-------------|
| **Transaction ID** | Unique identifier (e.g., TXN-001) |
| **Type** | ON-RAMP or OFF-RAMP badge |
| **Status** | Current status with color coding |
| **Amount** | Fiat and crypto amounts |
| **Accounts** | From ‚Üí To account flow |
| **Timestamp** | Date and time created |

Click any transaction to view complete details.

## Common Scenarios

<AccordionGroup>
  <Accordion title="On-Ramp stuck at 'Waiting for Funds'">
    **Likely cause**: Bank transfer not completed or not yet confirmed
    
    **Actions**:
    1. Verify you completed the bank transfer
    2. Check you included the reference code
    3. Wait full 2 hours for detection
    4. Contact support if still stuck
  </Accordion>
  
  <Accordion title="Transaction shows 'Failed'">
    **Check transaction details** for failure reason:
    - Compliance issue
    - Insufficient funds
    - Account problem
    - Technical error
    
    **Actions**:
    1. Read failure message
    2. Resolve the issue
    3. Try again or contact support
    4. Funds will be returned/refunded
  </Accordion>
  
  <Accordion title="Cannot find my transaction">
    **Solutions**:
    1. Check both Pending and Processed tabs
    2. Use search with transaction ID
    3. Verify you're logged into correct account
    4. Check All Transactions page
    5. Contact support with details
  </Accordion>
  
  <Accordion title="Transaction taking longer than expected">
    **On-Ramp** (Expected: 1-2 hours):
    - Wait 3 hours before contacting support
    - Check bank transfer was completed
    - Verify reference code was included
    
    **Off-Ramp** (Expected: 1-2 business days):
    - Wait 3 business days before contacting support
    - Weekends/holidays don't count
    - Check with your bank
  </Accordion>
</AccordionGroup>

## Transaction Lifecycle

### On-Ramp Timeline

```
[0 min] Transaction Created ‚Üí Click "Create On-Ramp"
[0-60 min] Waiting for Funds ‚Üí Complete bank transfer
[5-120 min] Processing ‚Üí Platform processes payment
[60-180 min] Completed ‚Üí Crypto in your wallet
```

### Off-Ramp Timeline

```
[0 min] Transaction Created ‚Üí Click "Create Off-Ramp"
[0 min] Processing ‚Üí Wallet debited, conversion starts
[24-48 hrs] Completed ‚Üí Fiat in your bank account
```
