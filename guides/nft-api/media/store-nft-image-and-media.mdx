## What is NFT Media Storage?

NFT media storage is where your actual image or media data will be hosted. The data in your NFT will link to this **image/media** and be displayed anywhere the NFT is used, such as in user wallets or on marketplaces.

## Upload NFT Images

Use this endpoint specifically for uploading NFT images. It automatically generates optimized versions for different display contexts.

### API Endpoint

<CodeGroup>
```javascript JavaScript
const formData = new FormData();
formData.append('file', imageFile); // imageFile is a File object

const response = await fetch('/api/v3/images', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: formData
});

const data = await response.json();
console.log('Image uploaded:', data);
```

```python Python
import requests

files = {'file': open('nft-image.png', 'rb')}

response = requests.post(
    '/api/v3/images',
    headers={'Authorization': 'Bearer YOUR_API_KEY'},
    files=files
)

data = response.json()
print('Image uploaded:', data)
```

```bash cURL
curl -X POST '/api/v3/images' \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -F 'file=@nft-image.png'
```
</CodeGroup>

## File Upload Validation

<CodeGroup>
```javascript JavaScript
const validateFile = (file) => {
  // Check file size (10MB for images, 100MB for videos)
  const maxSize = file.type.startsWith('image/') ? 10 * 1024 * 1024 : 100 * 1024 * 1024;
  if (file.size > maxSize) {
    throw new Error('File too large');
  }
  
  // Check file type
  const allowedTypes = [
    'image/jpeg', 'image/png', 'image/gif', 'image/svg+xml',
    'video/mp4', 'video/webm', 'audio/mpeg', 'audio/wav'
  ];
  
  if (!allowedTypes.includes(file.type)) {
    throw new Error('Unsupported file type');
  }
  
  return true;
};
```

```python Python
def validate_file(file_path):
    import os
    import mimetypes
    
    # Check file size
    file_size = os.path.getsize(file_path)
    mime_type, _ = mimetypes.guess_type(file_path)
    
    max_size = 10 * 1024 * 1024 if mime_type.startswith('image/') else 100 * 1024 * 1024
    
    if file_size > max_size:
        raise ValueError('File too large')
    
    # Check file type
    allowed_types = [
        'image/jpeg', 'image/png', 'image/gif', 'image/svg+xml',
        'video/mp4', 'video/webm', 'audio/mpeg', 'audio/wav'
    ]
    
    if mime_type not in allowed_types:
        raise ValueError('Unsupported file type')
    
    return True
```
</CodeGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="File upload failed">
    **Common causes:**
    - File size exceeds limits
    - Unsupported file format
    - Network connectivity issues
    - Invalid API key
    
    **Solution:** Validate file size and format before upload, check network connection and API credentials.
  </Accordion>
  
  <Accordion title="Slow upload speeds">
    **Possible causes:**
    - Large file sizes
    - Poor network connection
    - Server congestion
    
    **Solution:** Optimize file sizes, check network speed, try uploading during off-peak hours.
  </Accordion>
  
  <Accordion title="Media not displaying">
    **Common issues:**
    - Incorrect URL format
    - File corruption during upload
    - Browser compatibility issues
    
    **Solution:** Verify the returned URL, re-upload if necessary, test across different browsers.
  </Accordion>
  
  <Accordion title="File format not supported">
    **Supported formats:**
    - Images: JPG, PNG, GIF, SVG, WebP
    - Videos: MP4, WebM, MOV, AVI
    - Audio: MP3, WAV, OGG, M4A
    - Documents: PDF, TXT, JSON
    
    **Solution:** Convert files to supported formats before uploading.
  </Accordion>
