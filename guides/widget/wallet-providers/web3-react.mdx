## Overview

**web3-react** offers abstractions that facilitate the connection of your decentralized application (dApp) to web3 connectors. It also exposes methods for seamless interaction with these connections.

### Installation:

```javascript
npm install web3-react @venly/web3-react-venly
```

### Initialization:

```javascript
import { initializeConnector } from '@web3-react/core'
import { Venly } from '@venly/web3-react-venly';

export const [venly, hooks] = initializeConnector<Venly>(
  (actions) =>
    new Venly({
      actions,
      options: {
        clientId: 'YOUR_CLIENT_ID'
      },
    })
)
```
