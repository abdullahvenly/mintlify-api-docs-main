### Signature:

```javascript
{
    POPUP = 'POPUP',
    REDIRECT = 'REDIRECT'
}
```

### Parameters:

| Option     | Description                                                                                     |
| :--------- | :---------------------------------------------------------------------------------------------- |
| `POPUP`    | Open a popup to handle all signing and transactions                                             |
| `REDIRECT` | Navigate to connect to handle all signing and transactions, afterward redirect back to the page |
