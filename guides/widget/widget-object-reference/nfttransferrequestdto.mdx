### Signature:

```javascript
{
  walletId! : string;
  to! : string;
  secretType! : SecretType;
  tokenAddress!: string;
  tokenId!: number;
  data? : string;
  value! : BigDecimal;
  chainSpecificFields? : object;
}
```

  "from" : "0x89e01cEC55D718fA1Bf6B00fF21e6707cd9A3067",
