### Signature:

```javascript
{
   wallets: Wallet[],
   auth: KeycloakInstance,
   isAuthenticated: boolean
}
```

### Parameters:

| Parameter         | Description                                                             |
| ----------------- | ----------------------------------------------------------------------- |
| `wallets`         | Array of [`wallet`](https://venly.readme.io/docs/wallet)                |
| `auth`            | The [`KeycloakInstance`](https://venly.readme.io/docs/keycloakinstance) |
| `isAuthenticated` | Is true if the user is authenticated, false otherwise.                  |
