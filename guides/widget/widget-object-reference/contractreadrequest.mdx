### Signature:

```typescript
{
    secretType!: string,
    walletAddress!: string,
    contractAddress!: string,
    functionName!: string,
    inputs!: [
        {
            type!: string,
            value!: string
        }
    ],
    outputs!: [
        {
            type!: string
        }
    ]
}
```
