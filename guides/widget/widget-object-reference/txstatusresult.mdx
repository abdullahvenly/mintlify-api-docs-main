### Signature:

```javascript
{
    status: 'UNKNOWN' | 'PENDING' | 'FAILED' | 'SUCCEEDED'
}
```

### Parameters:

| Parameter | Type | Description |
|-----------|------|-------------|
| `status` | `String` | `UNKNOWN` Not yet visible on the chain. `PENDING` Not yet completed (or failed). `SUCCEEDED` Completed successfully. `FAILED` Transaction is failed. |


### Example:

```javascript
{
  "status" : "PENDING"
}
```
