### Signature:

```javascript
{
  walletId! : string;
  to! : SecretType;
}
```

> ðŸš§ Currently to only supports ETHEREUM, MATIC and BSC

### Parameters:

| Parameter  | Required | Type       | Description                                             |
| ---------- | -------- | ---------- | ------------------------------------------------------- |
| `walletId` | True     | String     | ID of the wallet one wants to import into another chain |
| `to`       | True     | SecretType | The chain you want to import the wallet into            |

### Example:

```json
{
  "walletsId" : "cdc4c08a-b8fa-4e4c-z5a2-92c87b80f174",
  "to" : "MATIC"
}
```
