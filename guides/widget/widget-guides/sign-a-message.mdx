# Intro

Signing a message is the equivalent to asking the user to sign his signature. Therefore to sign a message, we need to define the message for the user to sign and request his approval.

The object that can ask the user his approval is called a [`Signer`](https://venly.readme.io/docs/signer), once we've created a `Signer` object we can use the `Signer` to sign the message.

> ðŸš§ To perform actions on a wallet , such as transactions or signatures,** the wallet has to be linked to your application**.  The easiest way to link a wallet to your application in to use **[getAccount](https://docs.venly.io/docs/widget-user-authentication#get-user-account)** flow.  The benefits of using the getAccount flow is that this flow bundles multiple actions into one call.

### Function:

```javascript
//Asking the signer to sign a message.
venlyConnect.signer.signMessage({
    walletId: '<WALLET_ID>',
    secretType: '<BLOCKCHAIN>',
    data : "<YOUR_MESSAGE>"
})
```

### Example:

```javascript
venlyConnect.signer.signMessage({
    walletId: 'c8ec9954-fa1a-4682-9cf8-ef5c1015d1d1',
    secretType: 'ETHEREUM',
    data : "I agree with terms and conditions"
})
```
