# Introduction

To get the profile of a user you need to call the function [`getProfile`](https://venly.readme.io/docs/getprofile), which will return a promise including the user's information.

> ðŸ“˜ ðŸ§™ If you only want to retrieve a user's profile`getProfile` is a perfect function, but if you also need to retrieve his wallets and even authenticate the user, then it is recommended to [retrieve the user account](https://venly.readme.io/docs/retrieve-a-user-account).

### Function:

```javascript
venlyConnect.api.getProfile()
```

### Example:

```javascript
const venlyConnect = new VenlyConnect('YOUR_CLIENT_ID'); 
venlyConnect.api.getProfile().then((profile) => {
   console.log(`Users email, ${profile.email}, has been successfully executed!`);
});
```

### Returns:

```javascript
{
  userId: "4c56c8c3-faba-4475-9334-ff11f0280aed",
  username: "abdullah.baig@venly.io",
  email: "abdullah.baig@venly.io",
  firstName: "Abdullah",
  lastName: "Baig",
  nickname: "FulfilledBear",
  hasCustodialWallets: false
}
```
