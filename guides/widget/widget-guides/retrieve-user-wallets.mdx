# Intro

To fetch the wallet for a certain user you need to call the function [`getWallets`](https://venly.readme.io/docs/getwallets), which will return an array of [`wallets`](https://venly.readme.io/docs/wallet) with additional details for each wallet.

> ðŸ“˜ ðŸ§™ If you only want to retrieve a user's wallet then`getWallets` is the perfect function, but if you also need to authenticate the user and maybe get his email address or first name, then it is recommended to [retrieve the user account](https://venly.readme.io/docs/retrieve-a-user-account).

### Function:

```javascript
venlyConnect.api.getWallets();
```

### Example:

```javascript
const venlyConnect = new VenlyConnect('YOUR_CLIENT_ID'); 
venlyConnect.api.getWallets().then((wallets)=>{
   console.log(`The address of the first wallet is: ${wallets[0].address}`);
});
```
