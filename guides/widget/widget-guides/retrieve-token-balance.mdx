# Introduction

To fetch the balance of a certain token for a wallet, you need to call the function [`getTokenBalance`](https://venly.readme.io/docs/gettokenbalance) which will return a promise containing a [`TokenBalance`](https://venly.readme.io/docs/tokenbalance) object. If you need the entire balance a certain wallet holds, it is recommended to use the [`getTokenBalances`](https://venly.readme.io/docs/gettokenbalances) function. 

### Function:

```javascript
venlyConnect.api.getTokenBalance("<WALLET_ID>", "<TOKEN_ADDRESS>");
venlyConnect.api.getTokenBalanceByAddress("<SECRET_TYPE", "<WALLET_ADDRESS>", "<TOKEN_ADDRESS>");
```

### Example:

```javascript
const venlyConnect = new VenlyConnect('YOUR_CLIENT_ID'); 
venlyConnect.api.getTokenBalance("c8ec9954-fa1a-4682-9cf8-ef5c1015d1d1","0x7d1afa7b718fb893db30a3abc0cfc608aacfebb0").then((tokenBalance) => {
 console.log("The balance of",tokenBalance.symbol,"is",tokenBalance.balance);
})
```
