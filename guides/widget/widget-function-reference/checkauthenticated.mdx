This function call will check if the user is authenticated and redirect the user to `options.redirectUri` (if present) with the result.

```javascript
venlyConnect.checkAuthenticated();
```

> ðŸš§ If you set the [`redirectUri`](../object-type-reference/authenticationoptions.md#parameters) option, make sure that the SDK and the[`AuthenticationResult`](../object-type-reference/authenticationresult.md) handling is also present on the page you redirect to.

### Signature:

```javascript
venlyConnect.checkAuthenticated(options?: AuthenticationOptions): Promise
```

### Returns:

```javascript
Promise<AuthenticationResult>
```

### Parameters:

| Parameter | Type                                                                         | Required | Description                |
| --------- | ---------------------------------------------------------------------------- | -------- | -------------------------- |
| `options` | [`AuthenticationOptions`](../object-type-reference/authenticationoptions.md) | `False`  | Provide extra auth options |

### Example:

```javascript
// Redirect to https://wallet.venly.io
venlyConnect.checkAuthenticated({ redirectUri: 'https://wallet.venly.io' });
```
