This function transfers non-fungible tokens (ex. ERC721/ ERC1155 / VIP181/â€¦â€‹) from one address to another. The destination can be any blockchain address, a wallet, or a smart contract, it can even be an email address.

```javascript
venlyConnect.signer.executeNftTransfer({
    walletId: '71dec640-4eb8-4321-adb8-b79461573fc4',
    to: '0xf147cA0b981C0CD0955D1323DB9980F4B43e9FED',
    tokenAddress: '0x158b6a3540eeced8ecb40f9389e88f0902a3da9f'
    tokenId: '65'
    secretType: 'ETHEREUM',
})
```

### Signature:

```javascript
signer.executeNftTransfer(nftTransferRequestDto, options?): Promise<SignerResult>
```

### Returns:

```javascript
Promise<SignerResult>
```

### Parameters:

| Parameter                                                                     | Required | Description                                                                                                                                                                       |
| ----------------------------------------------------------------------------- | -------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [`nftTransferRequestDto`](https://venly.readme.io/docs/nfttransferrequestdto) | `True`   | The transfer request you want to execute. For more info on how this request should look like, see[ `nftTransferRequestDto`](https://venly.readme.io/docs/nfttransferrequestdto) . |
| [`options`](https://venly.readme.io/docs/redirectoptions)                     | `False`  | Redirect options you want to pass. Only available when using a REDIRECT [`signer`](https://venly.readme.io/docs/signer)                                                           |

### Example:

```javascript
venlyConnect.signer.executeNftTransfer({
    walletId: '71dec640-4eb8-4321-adb8-b79461573fc4',
    to: '0xf147cA0b981C0CD0955D1323DB9980F4B43e9FED',
    tokenAddress: '0x158b6a3540eeced8ecb40f9389e88f0902a3da9f'
    tokenId: '65'
    secretType: 'ETHEREUM',
}).then((signerResult) => {
    if (signerResult.success) {
        console.log(`Transaction ${signerResult.result.transactionHash} has been successfully executed!`);
    } else {
        console.warn(`Something went wrong while executing the transaction`);
    }
}).catch((reason) => {
    console.log(error);
});
```
