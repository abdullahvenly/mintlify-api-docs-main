This function returns the balance of all tokens currently supported by Venly. The list of supported tokens can be found on [Github](https://github.com/ArkaneNetwork/content-management/tree/master/tokens). Tokens need to be based on the ERC20 standard, this included for example ERC20, VIP180, TRC20, ...

```javascript
venlyConnect.api.getTokenBalances("d91b644a-076f-44bf-ae90-29251df19784");
```
