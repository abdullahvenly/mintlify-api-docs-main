This function signs arbitrary data using a specific account.

```javascript
venlyConnect.signer.signMessage({
    walletId: '71dec640-4eb8-4321-adb8-b79461573fc4',
    secretType: 'ETHEREUM',
    data : "I agree with terms and conditions"
})
```

### Signature:

```javascript
signer.signMessage(messageSignRequestDto, options?): Promise<SignerResult>
```

### Returns:

```javascript
Promise<SignerResult>
```

### Parameters:

| Parameter                                                                     | Required | Description                                                                                                                                                                 |
| ----------------------------------------------------------------------------- | -------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [`messageSignRequestDto`](https://venly.readme.io/docs/messagesignrequestdto) | `True`   | Signature request you want to sign. For more info on how this request should look like, see [`messageSignRequestDto`](https://venly.readme.io/docs/messagesignrequestdto) . |
| [`options`](https://venly.readme.io/docs/redirectoptions)                     | `False`  | Redirect options you want to pass. Only available when using a REDIRECT [`signer`](https://venly.readme.io/docs/signer)                                                     |

### Example:

```javascript
venlyConnect.signer.signMessage({
    walletId: '71dec640-4eb8-4321-adb8-b79461573fc4',
    secretType: 'ETHEREUM',
    data : "I agree with terms and conditions"
}).then((signerResult) => {
    if (signerResult.success) {
        console.log(`Successfuly signed: ${signerResult.result.signature}`);
    } else {
        console.warn(`Something went wrong while signing the message`);
    }
}).catch((reason) => {
    console.log(error);
});

```
