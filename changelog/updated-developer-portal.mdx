We are excited to announce the release of the updated Venly Developer Portal. This update brings a range of new features and improvements designed to enhance your experience and streamline your interactions with our services.

Below are the key highlights of this release:

## [Dashboard](https://portal.venly.io/overview)

### Requests

- Displays the number of API requests made in production and sandbox environments.
- Shows the number of errors.
- Click on **Details** for a comprehensive view of your API requests.

### Compute Units

- Shows the number of consumed and remaining compute units.
- Click on **Manage** to handle your subscription plan.
- Learn more about compute units [here](https://docs.venly.io/docs/pricing#compute-units).

![Portal Dashboard](https://files.readme.io/d6adb4a-image.png)

### Integrations

- Quick access to all Venly integrations.
- Click on each integration to view the documentation or getting started guide.

![Venly Integrations](https://files.readme.io/a73f76a-image.png)

## [Authentication](https://portal.venly.io/authentication)

### Access Credentials

- Provides access credentials for the widget and API services.
- Toggle between production and sandbox credentials.
- Learn more about authentication [here](https://docs.venly.io/docs/authentication).

### Endpoints

- View authentication and API service endpoints.
- Toggle between sandbox and production environments.
- Read about our environments [here](https://docs.venly.io/docs/environments).

![Authentication Page](https://files.readme.io/d198bd2-image.png)

### Login Events

- Track login events including date, time, and IP address.
- Monitor this section for any suspicious login attempts.

### Attack Mode

- Immediately stop all incoming traffic during a security issue.
- Activate to protect your platform from unauthorized access.
- Read our API security guide [here](https://docs.venly.io/docs/api-security-best-practices).

![Authentication Features](https://files.readme.io/c632511-image.png)

## Configuration

### Wallet API

- Create and configure webhooks.
- Enable or disable wallet exporting permission.
- Use the Matic and Avalanche faucet for the sandbox environment.
- More details [here](https://portal.venly.io/wallet-api/details).

### NFT API

- Create and configure webhooks.
- More details [here](https://portal.venly.io/nft-api/details).

### Widget

- Configure initialization and log-out URLs.
- Upload your company logo for the widget.
- Add a description about your company.
- More details [here](https://portal.venly.io/wallet-widget/details).

![Configure the Widget and API services](https://files.readme.io/9fa1d5d-image.png)

## [Usage](https://portal.venly.io/usage)

### Details

- Shows total number of requests, compute units consumed, and errors.
- Filter results based on time period and component (e.g., Authentication, Wallet API, NFT API).
- Learn how to track compute unit usage [here](https://docs.venly.io/docs/pricing#how-do-i-track-my-usage-of-compute-units).

![Track usage and compute units](https://files.readme.io/ef0e2e4-image.png)

We hope these updates improve your experience with Venly. Visit our [Developer Portal](https://portal.venly.io/overview).
