**BASE** and **XPLA** have been added to the supported blockchains in **Venly Widget**. Now you can create and manage XPLA/BASE wallets.

### Key Features

- Added support for **BASE** and **XPLA** blockchains in the Widget
- Create and manage wallets on **BASE** and **XPLA**, expanding the range of supported blockchains
- Send and receive funds to/from **XPLA** and **BASE** wallets directly through the Widget
- Seamless integration with applications using these blockchains
