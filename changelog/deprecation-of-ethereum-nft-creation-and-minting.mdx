title: "Deprecation of Ethereum NFT Creation and Minting"
description: "Ethereum NFT and ERC20 contract creation and minting no longer supported"

date: "2025-02-06"
---

Venly will no longer support **Ethereum** NFT creation and minting. This means that you will no longer be able to create new NFT contracts, ERC20 contracts, and mint tokens on Ethereum.

## Deprecation of Ethereum Minting - March 31st, 2025

<Warning>
**Effective Date: March 31st, 2025**

The following operations will no longer be supported on Ethereum:
</Warning>

<CardGroup cols={3}>
  <Card title="No New Contracts" icon="file-contract">
    Cannot deploy new NFT or ERC20 contracts on Ethereum
  </Card>
  
  <Card title="No Token Types" icon="layer-group">
    Cannot create new token-types on Ethereum contracts
  </Card>
  
  <Card title="No Minting" icon="coins">
    Cannot mint new NFTs or ERC20 tokens on Ethereum
  </Card>
</CardGroup>

<Info>
This decision has been made to align with our long-term product strategy and focus on supporting other blockchain ecosystems and technologies.
</Info>

## Affected API Services

The following API services will be impacted by this change:

- NFT-API v1
- NFT-API v2
- NFT-API v3
- Token-API

These APIs will no longer support Ethereum NFT creation or minting functionality. Please update your integrations accordingly.

## Affected Integrations

The following integrations will be impacted by this change:

- Zapier
- SiteManager

These integrations will no longer support Ethereum NFT creation or minting functionality.

## Documentation Updates

Our documentation has been revised to reflect this deprecation: a clear indication that Ethereum NFT creation and minting are no longer supported.

If you are currently using our services to create or mint NFTs on Ethereum, we recommend the following:
