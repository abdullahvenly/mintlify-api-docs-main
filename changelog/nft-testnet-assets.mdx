We're making it easier than ever to get started with **NFTs** with Venly! Every new **[Venly Portal](https://portal.venly.io/?utm_source=website&utm_medium=docs&utm_campaign=General+-+Venly+Internal)** user will now automatically receive an **NFT contract** with **three predefined token types** on the Polygon Amoy testnet.

## NFT - Testnet Assets

When a user creates a new company or signs up with [Venly Developer Portal](https://portal.venly.io/?utm_source=website&utm_medium=docs&utm_campaign=General+-+Venly+Internal), an **NFT contract** (Memberships example) is automatically created on the Polygon testnet chain.

### Predefined Token Types

  <Card title="Silver Membership" icon="medal">

  </Card>

### Minting Options

You can immediately mint NFTs to:
- **Wallet addresses**
- **Email addresses**

<Info>
If you mint an NFT to a user's email and they don't have a wallet, they will receive an onboarding link. Once they complete the onboarding process and create their wallet, the NFT will automatically appear in their wallet.
</Info>

## Accessing Testnet Assets

As soon as you sign up with Venly, navigate to the **NFT API** section in the [Portal](https://portal.venly.io/?utm_source=website&utm_medium=docs&utm_campaign=General+-+Venly+Internal) to see your pre-created NFT contract called **Memberships example**.

<Tip>
- Refresh the page or wait a few minutes after creating a new company for the contract to finish deploying
- Make sure the toggle is set to **Sandbox** on top right
</Tip>

## NFT Contract - Memberships Example

When opening the NFT contract, you can see the details:
- Contract address
- Chain
- Owner
- Royalty percentage
- Payout address

<Info>
Both the owner and payout address are set to the Polygon testnet wallet created for you. Access this wallet by navigating to the **Wallet API** section.
</Info>

## Minting NFTs

<Steps>
  <Step title="Choose Destination">
    Select whether to mint NFTs to a wallet address or an email address
    
    <Frame>
      <img src="https://files.readme.io/04c6663f06979cacc57e6ea51968af781dfe8fa2b7b95285887d5167feca6e23-image.png" alt="Minting Interface" />
    </Frame>
  </Step>
  
  <Step title="Specify Details">
    - Enter the destination address
    - Specify the number of NFTs to mint per address
    - Add multiple wallet/email addresses if needed
    - Click **Mint NFT**
  </Step>
  
  <Step title="Track Mint Status">
    You'll see a confirmation message and be redirected to track your mint requests
    
    <Frame>
      <img src="https://files.readme.io/ffa44fa71bdd44eea277a62a9aa61860a21ec7cf96095f5af91bdccc145bb48f-image.png" alt="Mint Status Tracking" />
    </Frame>
  </Step>
  
  <Step title="View Results">
    Refresh the page to see updated statuses. Click on a specific mint to view details
    
    <Frame>
      <img src="https://files.readme.io/39028fbf1e9a75a166179fa562e1fa8209d0904373112da048c8611a86181299-image.png" alt="Mint Results" />
    </Frame>
  </Step>
</Steps>
