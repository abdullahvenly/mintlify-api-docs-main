## NSFW Filter

Filter out your NFTs that are marked as **NSFW** using the new `includeNsfw` boolean query param that applies only to the following **three** endpoints:

<Steps>
  <Step title="Get NFTs by walletId">
    [API Reference](https://docs.venly.io/reference/listnonfungibles)
  </Step>
  
  <Step title="Get NFTs by Chain">
    [API Reference](https://docs.venly.io/reference/listnonfungibles_1)
  </Step>
  
  <Step title="Get NFTs by Chain and Wallet Address">
    [API Reference](https://docs.venly.io/reference/listnonfungiblesbyaddress)
  </Step>
</Steps>

### Response Includes isNsfw Parameter

You will receive a **contract.`isNsfw`** boolean in the response body for **all of the above endpoints** and the **following endpoints**:

4. Get NFTs by Chain and Contract Address: [reference](https://docs.venly.io/reference/getnonfungiblecontract)
5. Get NFTs by Chain, Contract Address, and `tokenId`: [reference](https://docs.venly.io/reference/getnonfungible)

### Parameter Explanations

<AccordionGroup>
  <Accordion title="includeNsfw Query Parameter" icon="circle-question">
    **How it works:**
    - The `includeNsfw` query param is set to **false** by default
    - If set to **false**, the response body will only return the **non-NSFW** results
    - If set to **true**, the response body will return all results (**NSFW + non-NSFW**)
  </Accordion>
  
  <Accordion title="isNsfw Response Parameter" icon="circle-info">
    **Response values:**
    - If `contract.isNsfw` is **true**, it indicates that your NFT collection is possibly **NSFW**
    - If `contract.isNsfw` is **false**, it indicates that your NFT collection is possibly **non-NSFW**
  </Accordion>
</AccordionGroup>

## Example Request

The following endpoint retrieves NFTs by chain and wallet address: [API Reference](https://docs.venly.io/reference/listnonfungiblesbyaddress)

```http
GET /api/wallets/MATIC/0x7312750DF4d2057b758a61C7017729c6Ec9bB3E9/nonfungibles?includeNsfw=true
```

### Query Parameters

| Parameter | Param Type | Description | Data Type | Value | Mandatory |
|:----------|:-----------|:------------|:----------|:------|:----------|
| `includeNsfw` | Query | Indication to include possible NSFW NFTs in the response body or not | Boolean | `true` | ‚ùå |
