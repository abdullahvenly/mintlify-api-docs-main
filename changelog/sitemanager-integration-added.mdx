Introducing the latest integration: **Sitemanager** which allows you to create NFT websites effortlessly. Create custom NFT contracts and allow users to mint custom NFTs directly from your website. The integration also includes token-gating.

## Build Your NFT Project with Sitemanager

Launched the core integration allowing users to create NFT projects directly through Sitemanager, leveraging Venly's API for streamlined NFT minting and management.

### Key Features

- **NFT Contract Creation**: Deploy custom NFT contracts with configurations, including the contract name, description, image, and metadata
- **Token Template Management**: Token templates enable project creators to establish predefined properties, enabling uniform structure across tokens
- **Custom Metadata for NFTs**: Support for adding custom metadata fields, making it easier to personalize tokens with project-specific details

## Multi-Tier Token Gating

Allows creators to set up tiered content access based on specific NFT collections, enabling multiple access levels within the same project.

<AccordionGroup>
  <Accordion title="Token Gating" icon="lock">
    Make specific content available only to users that hold NFTs from a specific NFT collection, helping create gated communities or offering exclusive resources/content to NFT holders
  </Accordion>
  
  <Accordion title="Button-Based Gating" icon="square-check">
    Configure to show or hide individual buttons based on NFT ownership
  </Accordion>
  
  <Accordion title="Block-Based Gating" icon="cube">
    Limit access to larger sections within a page, such as a pricing table, exclusive offers, or detailed tutorials. Choose to show or hide sections based on the NFTs an end user holds
  </Accordion>
  
  <Accordion title="Page-Based Gating" icon="file-lock">
    Restrict access to entire pages, making them accessible only to specific NFT holders. This can be ideal for premium content, gated communities, or exclusive event pages
  </Accordion>
</AccordionGroup>
