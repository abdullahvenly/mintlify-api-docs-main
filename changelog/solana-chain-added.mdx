### Key Capabilities

- **Wallet Management**: Create and manage wallets on the Solana blockchain
- **Sandbox Environment**: Works on the Solana Devnet
- **Fund Transfers**: Send and receive funds to/from Solana wallets
- **Token Support**: Transfer native SOL tokens, ERC20 tokens, and NFTs
- **Balance Queries**: Fetch balance of native tokens, ERC20 tokens, and NFTs
- **Program Execution**: Execute program calls on Solana
- **Message Signing**: Sign generic messages on Solana
- **Gas Estimation**: Retrieve the current gas fee
- **Read Programs**: Read program data
- **Wallet Import**: Import Solana wallets

## Create Solana Wallet

### Request

[API Reference](https://docs.venly.io/reference/createwallet)

```http
POST https://api-wallet-sandbox.venly.io/api/wallets
```

### Headers

| Parameter | Param Type | Value | Description |
|:----------|:-----------|:------|:------------|
| `Signing-Method` | Header | `id:value` | `id`: ID of the signing method<br/>`value`: Value of the signing method |

### Request Body

```json
{
  "secretType": "SOLANA",
  "description": "Solana Wallet",
  "userId": "eda7f8f6-2a70-4fcb-8a1f-387a88ae44d0"
}
```
