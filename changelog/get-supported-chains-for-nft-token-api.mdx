We have added two new endpoints for the NFT API and the Token API respectively. These endpoints allow you to fetch the list of supported chains for each API.

## Get Supported Chains for NFT API

<Card title="API Reference" icon="book" href="https://docs.venly.io/reference/getsupportedchainserc1155">
  View the complete API reference for getting supported chains
</Card>

### Request Endpoint

```http
GET /api/v3/erc1155/chains
```

### Response Body

```json
{
  "success": true,
  "result": [
    "AVAC",
    "BSC",
    "ETHEREUM",
    "MATIC",
    "HEDERA",
    "ARBITRUM"
  ]
}
```

<Card title="API Reference" icon="book" href="https://docs.venly.io/reference/getsupportedchainserc20">

GET /api/v3/erc20/chains
```

### Response Body

{

    "BSC",
    "ETHEREUM",
    "MATIC",

  ]

```
