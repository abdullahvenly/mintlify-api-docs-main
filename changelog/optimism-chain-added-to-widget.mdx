The **Optimism** chain has been added to the supported chains for the Widget. Now you can create and manage Optimism wallets using the **Widget**.

### Key Features

- Create and manage wallets on the **Optimism** blockchain
- Sandbox works on the **Optimism Sepolia Testnet**
- Send and receive funds to/from Optimism wallets through the Widget
- Transfer native tokens, ERC20 tokens, and NFTs on Optimism
- Execute contract calls on Optimism
- Sign EIP712 and generic messages on Optimism
- Read contracts and more!
