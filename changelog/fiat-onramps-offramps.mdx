Venly is excited to announce the release of **Fiat Onramp and Offramp API endpoints**, which allow you to generate on/offramp URLs for your users. This is available on all plans, including trials! We have integrated three providers: **Transak**, **Moonpay**, and **Ramp Network**.

## What's New?

Our new endpoints allow you to create **customized onramp and offramp URLs** for seamless integration into your platform.

## Configuring API Keys

### Sandbox Environment

<Tabs>
  <Tab title="Default Setup">
    Venly provides its own API keys for MoonPay, Transak, and Ramp Network in sandbox. These keys are ready to use for testing without any additional setup.
  </Tab>
  
  <Tab title="Customization">
    You can replace the default keys via the integrations section in the [Developer Portal](https://portal.venly.io/integrations) to test with your own credentials.
  </Tab>
</Tabs>

### Production Environment

<Warning>
**API Key Configuration Required**

When moving to production, you must configure your own API keys for MoonPay, Transak, and Ramp Network through the [Portal](https://portal.venly.io/integrations). Venly does not provide API keys for production.
</Warning>

## Create Customized URLs

<AccordionGroup>
  <Accordion title="1. Select Your Providers" icon="list-check">
    Choose from **Transak**, **Moonpay**, and **Ramp Network** based on your business requirements and regions you wish to serve.
  </Accordion>
  
  <Accordion title="2. Define Fiat and Crypto Options" icon="coins">
    Use parameters like `fiatCurrency` and `cryptoCurrency.symbol` to ensure transactions align with user preferences.
  </Accordion>
  
  <Accordion title="3. Set Transaction Defaults" icon="dollar-sign">
    Predefine parameters such as `fiatAmount` or `cryptoAmount` to streamline user interactions.
  </Accordion>
  
  <Accordion title="4. Establish Blockchain Network Preferences" icon="network-wired">
    Select blockchain networks using the `cryptoCurrency.network` parameter (e.g., Ethereum, Bitcoin).
  </Accordion>
  
  <Accordion title="5. Enable Localization" icon="globe">
    Use the `selectedCountryCode` parameter to tailor the experience for users based on their country.
  </Accordion>
  
  <Accordion title="6. Customize Redirect Flows" icon="arrow-right">
    Set a `redirectUrl` to direct users back to your platform after transactions.
  </Accordion>
  
  <Accordion title="7. Streamline User Onboarding" icon="user-plus">
    Pre-fill parameters like `email` and `walletAddress` to reduce friction and speed up transactions.
  </Accordion>
</AccordionGroup>
