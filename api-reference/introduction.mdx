## Getting Started

All Venly API endpoints require authentication using API keys. You can obtain your API keys from the [Venly Console](https://portal.venly.io/).

### Authentication

All API endpoints are authenticated using Bearer tokens in the Authorization header:

```bash
curl -X GET "https://api.venly.io/api/endpoint" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json"
```

### Base URLs

- **Production**: `https://api.venly.io`
- **Staging**: `https://api-staging.venly.io`

### Rate Limits

API requests are subject to rate limiting to ensure fair usage across all users. Rate limit information is included in response headers.
